Usage Issues

The API documentation presented in the link above provides a slightly misleading interpretation of how to use the functions and classes in DST and SOM from a Python interpreter session or from within a Python script/module. From the documentation, one might get the impression that the proper usage is: 

 >>> SOM.som.SOM

or 

 >>> SOM.som

which is package_name.file_name.function_name or package_name.file_name. However, due to the way the package is configured to install, the correct usage is: 

 >>> SOM.SOM

No file name is present in the correct usage scenario.

The SNS NeXus API is accessed via the sns_napi module.
